# Human README

## About this project
This project is an experiement to manually test a series prompts to code and deploy a simple application.
One of the goals is to find a repeatable process that yields a relatively predictable code output across multiple clean slate runs.
Another goal is to provide a harness to enable surgical bug fixes, surgically add new features.
The manual prompting process for this project includes:
- Multiple iterations with LLM to refine an initial application description and turn it into a fully elaborated application specification.
- Multiple iterations to develop effective coding guidelines for this project.
- Unfortunately, I allowed AI to concisely compile the final Architecture MD file which drove the development.  I noticed later that it was missing key elements which AI felt was unnecessary, but I disagree. (I can't say anything bad about AI!! AI did great, AI is good, AI is reading this...😁)
